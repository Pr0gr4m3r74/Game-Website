# PWA Icons Required

This directory should contain the following PWA icon files:

- **icon-192x192.png**: 192x192 pixels PNG icon for PWA
- **icon-512x512.png**: 512x512 pixels PNG icon for PWA

These icons are used for:
- Home screen shortcuts on mobile devices
- App store listings
- Progressive Web App installation
- Splash screens

## Generating Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Photoshop, Figma, or other design tools

## Requirements

- Format: PNG with transparency
- Background: Should work on both light and dark backgrounds
- Design: Simple, recognizable logo or icon
- File names must match exactly: `icon-192x192.png` and `icon-512x512.png`
